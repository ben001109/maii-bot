


# 下一階段工作計劃（Next Time Plan）

## 目標
建立並完善遊戲循環模組，實現核心的遊戲時間推進、冷卻計時與收益計算功能，為後續擴充任務、研發及排行榜系統打好基礎。

---

## 工作項目

### 1. 實作遊戲循環架構
- 建立事件總線 (Event Bus) 模組，用於發布與訂閱遊戲內事件
- 實作遊戲時間推進機制（例如 1秒現實 = 6秒遊戲時間）
- 設計並實現遊戲主循環 `tick`，觸發冷卻與收益計算事件

### 2. 玩家冷卻系統
- 在玩家資料中新增冷卻相關欄位及初始化
- 設計冷卻計時器的遞減邏輯
- 實現冷卻結束通知機制（可發送 Discord 私訊或頻道提示）

### 3. 企業收益計算
- 抽象收益計算邏輯，支持不同企業類型的收益策略
- 依照遊戲時間，自動計算並累積企業收入
- 收益入帳機制及資料庫同步

### 4. 模組整合與測試
- 整合遊戲循環模組與現有玩家及企業服務
- 撰寫單元測試與整合測試，確保遊戲時間與收益邏輯準確
- 日誌與錯誤處理完善，方便後續調試與維護

### 5. 後續擴展預留
- 任務系統事件介面設計  
- 研發系統相關事件與資料結構規劃  
- 排行榜資料抓取與刷新機制

---

## 注意事項
- 優化事件處理流程，避免阻塞主執行緒  
- 玩家資料冷卻與收益資料需與 Redis 及 PostgreSQL 同步一致  
- 撰寫清楚的開發文件與流程圖，方便團隊協作
