# Maii Bot

This is a minimal Node.js project with ESLint and Prettier configured.

## Commands

- `npm run lint` – run ESLint on the project files.
- `npm run format` – format the files using Prettier.
- `npm test` – run the simple test script.

## Project Structure

```
.

├── .prettierrc
```
